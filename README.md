# 2025Growthathon_Team8

# 📌 **Plogging Buddy**
> 환경 보호를 위한 플로깅(Plogging) 모임 찾기 플랫폼입니다.

---

## 🚀 **Features**
- 주요 기능 1️⃣
  사용자 위치 기반으로 플로깅 모임 조회 및 참여 (CRUD)
- 주요 기능 2️⃣
  플로깅 모임 개설 및 참여 신청 기능
- 주요 기능 3️⃣
  카카오 지도 API를 활용한 모임 장소 표시
- 주요 기능 4️⃣
  카카오 공유 API를 활용한 SNS 공유 기능

---

## 🛠️ **Tech Stack**
- Frontend: React.js
- Backend: Spring Boot
- Database: 
- API: Kakao Map API, Kakao Share API
- Tools: Git, VSCode
